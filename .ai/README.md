# AI Context Directory

Context files for AI coding assistants working on Just In Time.

## Files

| File | Purpose |
|------|---------|
| `CONTEXT.md` | Project overview and current state (start here) |
| `config.yaml` | AI behavior configuration |
| `specs/SPEC.md` | Product specification |
| `architecture/ARCHITECTURE.md` | System architecture with diagrams |
| `architecture/PATTERNS.md` | Code patterns and conventions |
| `decisions/` | Architecture Decision Records |
| `prompts/` | Reusable prompt templates |

## Usage

Point your AI assistant to `.ai/CONTEXT.md` for full project context.
